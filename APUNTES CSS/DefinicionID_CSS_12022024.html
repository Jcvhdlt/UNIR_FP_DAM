Elementos de Selección en CSS: ID y CLASS

Características de los elementos ID en CSS

Un ID en CSS, se conoce como un identificador único.
Quiere decir, que el valor del atributo id es único para un elemento o etiqueta el cual vamos a aplicar en HTML.
El id se representa en HTML como id="nombre_identificador" y en CSS con #nombre_identificador.
