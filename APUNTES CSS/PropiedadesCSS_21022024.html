Colores en CSS

Las propiedades que podemos utilizar en CSS para aplicar colores son las siguientes:
  • color: Cambiar el color del texto en css
  • background-color:Cambiar el color de fondo de un elemento

A parte de esto, debemos tener en cuenta que podemos poner de varias maneras colores en CSS:
  • Palabra Clave: Por ejemplo: color:red; background-color:grey; El propio nombre del color
  • Esquema rgb(red, blue, green): rbg(255,255,48)
  • Esquema rgba(red, blue, green, alpha): rgba(255,255,48,valores del 0-1) donde 1 es la capa más oscura, y el 0 la más clara
  • Esquema RGB Hexadecimal: #RRBBGG por ejemplo: #FF0055
  • Esquema RBG Hexadecimal Abreviado:#RBG por ejemplo: #F05

Unidades en CSS: Absolutas y Relativas

  • Unidades Absolutras: Son aquellas que tienen una medida fija.
    Son unidades que son poco flexibles y adaptables a resoluciones para diferentes tamaños de pantallas.
    Por: cm, picas, in (pulgadas), puntos (pt)...
  • Unidades Relativas: Son aquellas que tienen una medida más flexible y adaptable a las resoluciones, y por lo tanto,
    son las que se suelen utilizar por lo general para la programación / maquetación web.
    em (16px), rem, px (pixel), % (porcentajes)...
  • width: para la anchura de nuestro elemento
  • height: para la altura de nuestro elemento

Modelos de Cajas en CSS

Todos los elementos HTML, cuando creamos páginas web, están dentro de lo que se conoce como cajas o contenedores de contenido.
Las cajas que vamos a utilizar, son principalmente div (cajas) o etiquetas semánticas como pueden ser: header, nav, footer,
aside, article...
Hasta ahora, utilizabamos tablas para maquetar, pero estas son poco flexibles a la hora de poderlas ajustar a distintas resoluciones,
y por lo tanto, a la hora de realizar maquetación web lo que vamos a utilizar será este tipo de cajas (div) con una serie de propiedades
que son las que se comentarán a continuación:
  • Margin:Corresponde al margen existente entre el borde exterior de nuestra caja/elemento contenedor y nuestro navegador
  • Padding:Corresponde al margen interior de nuestra caja, es decir, el espacio existente entre el contenido y el borde
    de nuestro elemento contenedor
  • Contenido:Son las etiquetas que van a ir dentro de nuestra caja o elemento contenedor

Elementos en Línea y Elementos en Bloque

  • Elementos en línea: son aquellos que se van sucediendo a lo largo de la misma línea de nuestra web, mientras vayan
    cogiendo unos detras de otros y de izquierda a derecha.Van por lo tanto siguiendo el flujo de de la orientación
    asignada.
  • Elementos en bloque: son elementos que, independientemente de la anchura que tengan, se separan verticalmente de los Elementos
    posteriores y anteriores. Es como si provocaran un salto de línea antes y después de un elemento.

Modelos de cajas en CSS: Zonas de un elemento

Dentro de una caja podemos encontrarnos las siguientes zonas a tener en cuenta:
  • Top: se refiere a la parte superior del elemento
  • Lef: se refiere a la parte izquierda del elemento
  • Right: se refiere a la parte derecha del elemento
  • Bottom: se refiere a la parte inferior del elemento
  • Center: que hace referencia a la posición central entre dos elementos


Esto elementos podemos aplicarlos de la siguiente manera:
  • margin-top o padding-top
  • margin-bottom o padding-bottom
  • margin-right o padding-right
  • margin-left o padding-left

La aplicación de estos elementos se pueden establecer de la siguiente manera:
  • margin con un solo valor: aplica a los 4 márgenes de los elementos: P.E. margin:40px
  • margin con dos valores: aplica el primer valor al top/bottom, y el segundo al right/left. P.E: margin: 4px 20px
  • margin con tres valores: aplica el primer valor al margen top, segundo valor a right/left, y el tercero al bottom.
    P.E. margin: 4px 20px 5px.
  • margin con cuatro valores: aplica el primero al top, segundo al right, tercero al bottom y el cuarto al left.
    P.E. margin: 4px 6px 20px 8px.

MUY IMPORTANTE: Cuando se establecen estos valores, siempre se tiene que tener en cuetna que se sigue el sentido de las agujas
del reloj. Siendo en este caso, el primer elemento el top, segundo el right, tercero el bottom y cuarto el left.
Que esto no quita que podáis aplicar un valor a un lado concreto.P.E. margin-top: 20px y se aplicaría un margen de 20px sólamente al top.

Tipos de Bordes en CSS

CSS nos permite utilizar diversos tipos de propiedades para utilizar en los bordes de cajas, imágenes y otros elementos. Para ello podemos utilizar la siguiente propiedad:
  • border-color: se utiliza para poder dar color a un borde.
  • border-width: es una propiedad que se utiliza para dar grosor al borde. thin|medium|thick
  • boder-style: define el estilo del borde en CSS
Los tipos de border-style que podemos encontrarnos son los siguientes:
  • none: no aplica ningún tipo de estilo y coge el valor por defecto.
  • hidden: "oculta" los bordes
  • dotted: establece puntos en los bordes
  • dashed: establece rayas discontinuas
  • solid: borde sólido
  • double: doble línea de borde
  • groove: biselado con luz desde arriba
  • ridge: biselado con luz desde abajo
  • inset: establece un borde con profundidad hacia dentro
  • outset: establece un borde con profundidad hacia fuera
