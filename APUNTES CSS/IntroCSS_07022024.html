Introducción a CSS

Hasta ahora, cuando hemos diseñado páginas web, siempre hemos mezclado la parte de información/estructura (HTML) con la parte más
visual de apariencia: colores, fondos, tamaños de tipografía, tipos de letra, etc.
Actualmente, esto ya no se aplica de esta manera, sino que se tiene por un lado la parte de información / estructura recogida
en un documento HTML, y por otro lado, tenemos toda la parte visual / apariencia en un documento denominado Hoja de Estilo
o CSS ( Cascading Style Sheet)
El motivo por el que se tiene separado la parte de HTML de la parte de CSS, es para poder tener todo mucho más organizado y sobre todo,
poder localizar todo el contenido de una manera más rápida y sencilla en caso de que se tenga que modificar cualquier tipo de información /
apariencia de nuestro proyecto web.
Ejemplo de lo que se entiende como HTML y lo que se entiende como CSS

En conclusión, podemos establecer que HTML definirá la estructura semántica y contenido de nuestra web, mientras que CSS, definirá, como es
nuestra página web en cuanto a apariencia: tamaño de letra, color, márgenes, posicionamiento, etc.
Podemos usar un único CSS para varios archivos HTML, puesto que en este último es donde decidiremos qué tipo de CSS queremos aplicar.
La extesión de archivo HTML es .html, mientras que la extensión de archivo CSS es .css. Esto es siempre y cuando vayamos a utilizar
un documento CSS externo.

Tipos de CSS: Cómo usarlos?

Antes de empezar con CSS, tenemos que conocer que podemos usarlo de tres maneras o formas distintas:
  • CSS Externo:Un CSS Externo, es un archivo con extensión .css. Se vincula al HTML en el Head, a través de la etiqueta link.
    link rel = "stylesheet" type = "text/css" href= "style.css"
  • CSS Interno: Es un CSS que se establece mediante las etiquetas style de apertura y cierre dentro de mi head
  • CSS Embebido / Incrustrado: Un CSS que se aplica directamente en la etiqueta (tag) de HTML, a través del atributo style
