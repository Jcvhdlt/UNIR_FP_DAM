Características de los elementos CLASS en CSS

Un Class en CSS, se conoce como un atributo el cual puede usarse para distintos elementos de distintas familias en HTML.
Tiene la parte positiva o buena de que cuando tenemos que aplicar los mismos estilos a diferentes elementos en HTML,
nos ayuda a reducir las líneas de código de nuestro .css (externo) o css (interno)
El class en HTML se representa como class="nombre_identificador" y en CSS con .nombre_identificador
