Propiedad de Profundidad de Niveles (Z-INDEX)

La propiedad z-index, es un elemento que podemos usar en CSS para establecer Profundidad
en un elemento sobre el resto.

De esta manera, lo que nos permite es que podemos situar un elemento (principalmente se usa con
imágenes) una encima de otra.

Su funcionamiento por lo tanto es muy sencillo:
Para poder establecer dicha profundidad, sólo tenemos que indicar el nivel de profundidad del elemento:
es decir, si le damos un valor positivo (mayor de 0), estamos situando los elementos por encima del resto.

Si le damos un valor negativo (menos de 0), estamos situando los elementos por debajo del resto.

Si no le damos un valor, es decir, por defecto se toma el valor 0, el elemento se queda tal y como se va a mostrar.

IMPORTANTE: La capa o nivel 0, es la capa desde la que partimos, es decir, es la que viene por defecto.

Además, es la que nos va a guiar para establecer si un elemento está por encima por debajo del resto.

Es importante también indicar, que los niveles z-index, así como las propiedades left, top, right y bottom,

NO FUNCIONAN con elementos que están utilizando posicionamiento STATIC (Estático). Su posicionamiento
deberá ser distinto a este.
