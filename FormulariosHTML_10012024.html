Formularios en HTML

Los formularios en HTML, son elementos que nos van a dotar de cierta funcionalidad y dinamismo a nuestras páginas web.
Nos van a permitir interactuar con nuestras páginas web y por lo tanto son elementos fundamentales para poder recoger información
y posteriormente almacenarla con el fin de gestionar esta.
Los formularios permiten dos tipos de funciones:
  • Recoger información del usuario
  • Enviar esos datos que se han guardado, para procesarlos o realizar cualquier tipo de operación que se requiera con ellos.

Los principales objetivos que tenemos que tener en cuenta a la hora de diseñar un formulario en HTML, son los siguientes:
  • El formulario tiene que ser lo más sencillo posible a la hora de la inserción de datos por parte del usuario.
  • La experiencia del usuario (UX) sea lo más agrable posible.
  • Que los datos introducidos por el usuario estén en un formato predecible y esperado
  • Reducir los errores al introducir datos en la medida de lo posible y comunicar cláramente al usuario si esto ocurre.

Etiquetas básicas para la creación de un Formulario en HTML

Para crear un formulario en HTML, utilizaremos la etiqueta con apertura y cierre llamada form
A parte de la etiqueta form que establece que vamos a usar un formulario en HTML, tenemos que tener en cuenta una serie de propiedades
que van asociada a estas:
  • action: Indicar la dirección URL o BBDD o web, etc. (Dirección Backend) a la que queremos enviar la información de nuestro
    formulario.
  • method: Método puede ser get o post. Get se utiliza cuando queremos mostrar la información que se envía
    de nuestro formulario a través del protocolo HTTP/HTTPS de nuestra web.Post se utiliza cuando queremos oclutar la información
    de nuestro formulario a través del protocolo HTTP/HTTPS
  • name: dar un nombre del formulario.
  • target: Se utiliza para abrir el formulario en una pestaña nueva
  • enctype:Codificar la información del envio del formulario.
  • autocomplete: Tiene función on u off, es para activar o desactivar el autocompletado de nuestro formulario en la web.

Tipos de Datos que podemos gestionar en un Formulario de HTML

Podemos obtener cualquier tipo de información que se quiera/requiera de un formulario. Sí que es verdad, que los más utilizados son
los siguientes:
  • Información de texto: Nombres, apellidos, direcciones físicas, etc. Utilizaremos elementos input y textarea (type=text).
  • Número o cantidades: Edades, precios, cuantías, etc. Utilizaremos elementos input pero para números (type=number)
  • Fechas u horas. Fecha de nacimientos, calendarios, etc.Utilizaremos elementos input pero para fechas (Eje:type=date)
  • Verdadero o Falso. Utilizaremos input type=checkbox
  • Opciones únicas donde podemos elegir entre 2 o + opciones. Utilizaremos input type=radio o select
  • Varias Opciones. Utilizaremos select multiple, input checkbox
  • Opciones abiertas. datalist
  • Seleccionar colores: input type=color
  • Seleccionar archivos. inpunt type=file
